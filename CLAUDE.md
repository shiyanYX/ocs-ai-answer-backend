## Agent skills

### Issue tracker

Issues 位于 GitHub Issues，使用 `gh` CLI 操作。详见 `docs/agents/issue-tracker.md`。

### Triage labels

使用五标签标准词汇：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`（`wontfix` 已存在于仓库）。详见 `docs/agents/triage-labels.md`。

### Domain docs

单上下文项目：根目录 `CONTEXT.md` 定义领域术语，`docs/adr/` 存放架构决策。详见 `docs/agents/domain.md`。
